# Exercises

This folder is where you’ll do the hands-on practice for this module. Each exercise includes a **Starter** version (what you begin with) and a **Solution** version (a completed reference).

## How these exercises are organized

Each exercise lives in its own folder, typically named like:

- `exercise1-some-topic`
- `exercise2-some-topic`

Inside every exercise folder, you will always find:

- `starter/`  
  Your starting point. This is the version you should open first, edit, and run.

- `solution/`  
  A completed version you can use to check your work after you’ve made an attempt.

## Typical structure

    exercises/
      exercise1-some-topic/
        starter/
        solution/
      exercise2-some-topic/
        starter/
        solution/
      README.md

## Recommended workflow

1. Open the exercise folder for the activity you’re on (for example, `exercise1-*`).
2. Start in `starter/` and follow the instructions provided in the exercise or module README.
3. Run the code, complete the steps, and review your outputs.
4. After you finish (or if you get stuck), open `solution/` to compare your work and understand what changed.
